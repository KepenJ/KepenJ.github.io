标准库和 std 命名空间

C++ 中也沿用了 C 语言中的标准库，引用方式也还是 include <XXX> 的方式，下图就是常用的库

同时 C++ 也衍生了一些新的类如

  • iostream.h //输入输出头文件。
  • fstream.h //文件操作头文件。
  • wchar.h //宽字符处理头文件。

而最新的 C++ 库标准我们已经可以使用 include <iostream> 这种方式来引入，你会发现我们引入的是 iostream 而非 iostream.h ，这样新的引入方式将这些类、方法引入到一个默认的命名空间中 std。 同样针对 C 语言的库你可以通过 <string.h>—><cstring> 、 <stdio.h>—><cstdio> 等演变来实现同样的功能。

输入输出

这个是 C++ <iostream> 库中的功能，C++中使用 cout 最为输出关键字，cin 作为输入。当然 C 语言的 scan 和 printf 之类的也依然适用。只是这样的改写和封装，使得输入和输出更为直接明了和方便。

新增数据类型

相比 C 语言中的数据类型，C++中增加了 long long、bool 和 wchar_t 这三种类型。

其中 long long 是 C99 新增的一种数据类型，用于支持64位整数。由于 C99 标准发布较晚，所以各种 C/C++ 编译器对 long long 的支持比较混乱，C/C++中long long与__int64的区别。
